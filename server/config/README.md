# Config

Create a file here called `private_config.js`

Fill it in this style, but with each field filled in.

This file will be ignored by git, and is local to your machine.

```js
module.exports = {

  // ===========================================================================
  // Twitter API Keys ==========================================================
  // ===========================================================================
    twitter: {
      consumer_key: '',
      consumer_secret: '',
      token: '',
      token_secret: ''
  }

};
```
